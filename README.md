## I want to create a high school fitness application with a multi-tier architecture: presentation, logic, and data layers.

Requirements:
1. Presentation Layer:
   - Responsive web dashboard
   - Pages: Student dashboard, Teacher dashboard, Login
   - Display student progress, exercises, and charts

2. Logic Layer:
   - Handle authentication for students and teachers
   - Track exercises and scores
   - Assign workouts based on student performance
   - Validate user input

3. Data Layer:
   - Database schema for Students, Teachers, Classes, Workouts, Scores
   - CRUD operations for each entity
   - Use Node.js with Express and a relational database (SQLite or PostgreSQL)

Please generate:
- Folder structure separating presentation, logic, and data layers
- Starter files for each layer
- Basic routing and sample data for demonstration
